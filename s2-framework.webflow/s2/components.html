Modal (Pop-up Dialog)

Flexible: Place the dialog modal modals anywhere on a page.

Attributes: Corresponding modal open by matching the data-modal="modal-id" attributes.

Simple: Add a is-open class to open up the dialog for easy editing.

Accessible: Leverage accessibility features of native dialog.

Unlock advanced features by adding the below attributes to any dialog modal modal:

  • data-modal-open="2500ms": trigger to open after certain "s" or "ms"
  • data-modal-open="800px": trigger to open after scrolling down certain "px"
  • data-modal-reopen="3": the trigger will function again X days after closing a modal
